# Scholar's Horizon – Computer Science Research App

## Overview
A modern Flutter application to explore Computer Science research papers from arXiv with a clean UI, smart search, and multi-category filtering.

## Features
- Computer Science-focused approach using arXiv
- Multi-select category filtering using checkboxes
- Smart search by author, title, keywords, and subject
- Real-time suggestions
- Fast & reactive UI using Flutter + Riverpod

## Supported Categories
- Artificial Intelligence (cs.AI)
- Machine Learning (cs.LG)
- Computer Vision (cs.CV)
- Distributed Systems (cs.DC)
- Cybersecurity (cs.CR)
- Human-Computer Interaction (cs.HC)

## Architecture
```plaintext
lib/
 ├── providers/
 ├── models/
 ├── services/
 ├── widgets/
 └── pages/
```

## API Used
arXiv API: http://export.arxiv.org/api/query
Getting Started
1. Install Flutter SDK
2. Clone repository
3. Run flutter pub get
4. Run flutter run
